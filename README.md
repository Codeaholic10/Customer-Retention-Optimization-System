## Customer-Retention-Optimization-System
An End-to End Customer Segmentation and Churn Prediction Project

##Overview

##Objectives

Build an end-to-end ML system using the Telco Customer Churn dataset that:
1.Performs customer segmentation using clustering.
2.Trains a global churn prediction model.
3.Trains separate churn models per segment.
4.Implements a profit-aware targeting engine that ranks customers based on expected retention value under budget constraints.
5.Exposes functionality via FastAPI endpoints.
6.Is modular, clean, and production-structured.

##Why This Project Matters

Many churn prediction projects stop at training a classification model.
This project goes further by:

1.Incorporating customer segmentation
2.Enabling segment-aware modeling
3.Preparing for profit-aware decision optimization

The long-term goal is to transform churn prediction into a business decision system rather than a simple predictive model.
